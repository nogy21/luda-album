# Luda Album Photo-First Product Principles

## 제품 원칙

`루다 앨범 = 사진 탐색 속도 + 회상 경험`

핵심 경험에서 덕담은 보조 기능으로 두고, 메인 사용 흐름은 사진 탐색에 집중한다.

## 필수 UX 축

1. 월/연 아카이브 네비게이션
- 월별 카드로 빠르게 점프할 수 있어야 한다.

2. 무한 스크롤 + 지연 로딩
- 초기 로딩은 가볍게 유지하고, 사용자가 내려갈 때 점진적으로 더 불러온다.

3. 대표컷/하이라이트 분리
- 처음 방문한 사용자도 바로 감정 포인트를 느끼게 한다.

4. 라이트박스 탐색 연속성
- 이전/다음 이동이 자연스럽고, 닫아도 원래 위치를 잃지 않는다.

5. 성능 최적화
- `next/image` 최적화, 썸네일/원본 전략, 캐시 정책을 기본값으로 둔다.

## CTA 우선순위

1순위: `이번 달 사진 보기`  
2순위: `루다 성장 타임라인 보기`

## 적용 상태

- `/photos` 히어로 CTA를 사진 중심 우선순위로 반영
- 월별 아카이브 빠른 점프 칩 추가
- 월별 확장 영역에서 sentinel 기반 자동 추가 로딩 적용
